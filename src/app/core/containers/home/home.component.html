Welcome to Play Arkham World
This site contains a digital version of "Arkham Horror: The Card Game" by Fantasy Flight Games.
Disclaimer
Play Arkham World is developed as a hobby project in spare time. It is intended to run on a 1920 pixels wide screen (preferably in full screen (hit F11)).
The game is not suitable for Mobile devices. Game progress is stored in your browsers local storage and cannot be transferred between browsers or devices. Please send bug report or ideas to alittlebitofcode@gmail.com
